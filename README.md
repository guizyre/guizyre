### 👨‍💻 Sobre mim

Sou um grande entusiasta da área de tecnologia, com forte interesse tanto no desenvolvimento de software quanto na interação de baixo nível com o hardware. Gosto de entender como as coisas funcionam desde a arquitetura física até o código em produção.

- 🎓 **Formação:** Estudante de Engenharia da Computação no **IFPB** (Instituto Federal da Paraíba).
- ☕ **Desenvolvimento de Software:** Foco principal em Java (POO), com ótimos conhecimentos em Python, C e C++.
- 🌐 **Web e Banco de Dados:** Experiência com Banco de Dados Relacionais (MySQL), desenvolvimento web (HTML5, CSS3) e configuração de servidores web (WampServer e Apache Tomcat).
- ⚙️ **Hardware e Sistemas:** Conhecimentos sólidos em **Organização e Arquitetura de Computadores (OAC)** e experiência prática com programação de microcontroladores, como a placa **STM32F407VET6**.
- 🚀 **Objetivo:** Sempre buscando aprender novas arquiteturas e resolver problemas complexos com código limpo e eficiente.
